# Freshway Market

> **"The Modern Green Grocer"** — A premium grocery website portfolio project

## 🚀 Quick Start

```bash
# 1. Install dependencies
npm install

# 2. Run development server
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 What's Included

- **Next.js 14** with App Router
- **React Server Components** for optimal performance
- **CSS Modules** with design tokens
- **Zustand** for state management (cart)
- **React Three Fiber** for 3D product inspection
- **Lenis** for smooth scrolling
- **GSAP** for animations
- **Fully responsive** mobile-first design
- **Accessible** with reduced-motion support

## 📁 Project Structure

```
freshway/
├── app/                   # Next.js App Router pages
│   ├── page.tsx          # Home
│   ├── shop/             # Shop with filters
│   ├── product/[slug]/   # Product details
│   ├── cart/             # Shopping cart
│   └── about/            # About page
├── components/
│   ├── core/             # Reusable UI components
│   ├── layout/           # Header, Footer, Cart
│   └── sections/         # Page sections
├── styles/               # Global styles & tokens
├── lib/                  # State & hooks
├── data/                 # Mock product data
└── public/              # Static assets
    └── assets/           # vid.mp4 & model/tray.gltf
```

##  Features

- ✅ **Video Hero** — Atmospheric background video
- ✅ **3D Product Viewer** — Interactive product inspection
- ✅ **Shopping Cart** — Zustand-powered state management
- ✅ **Product Filtering** — Category and price filters
- ✅ **Responsive Design** — Works on all devices
- ✅ **Premium Animations** — Smooth micro-interactions
- ✅ **Accessibility** — WCAG compliant

## 🎨 Design System

The project uses a comprehensive design system with:
- **Color palette** — Forest green, Zest orange, Cream
- **Typography** — Outfit (headings) + Inter (body)
- **Spacing scale** — Consistent spacing tokens
- **Motion** — Calm, purposeful animations

## 📝 Notes

- This is a **portfolio/demo project** — no backend or payment processing
- Product data is **mocked** in `data/products.json`
- Assets (video & 3D model) should be in `public/assets/`
